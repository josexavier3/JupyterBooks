# Resolução de Problemas em Engenharia Mecânica usando Jupyter Notebooks

Recurso educativo aberto com **resoluções de problemas** das unidades curriculares de
Engenharia Mecânica lecionadas no Departamento de Engenharia Mecânica e Industrial (DEMI) da
Faculdade de Ciências e Tecnologia | FCT NOVA.

Cada problema tem a resolução em **Jupyter Notebook** (Python), que pode ser:
- **aberta no browser**, a correr sem instalar nada (JupyterLite), ou
- aberta no Google Colab.

## Unidades curriculares

- **MSI** — Mecânica dos Sólidos I
- **MSII** — Mecânica dos Sólidos II
- **MAI** — Mecânica Aplicada I
- **ProjFLMEA** — Projeto de Fadiga de Ligações Mecânicas e Estruturas Aeroespaciais

O site publicado (landing + um livro por UC + execução no browser) é gerado com
[Jupyter Book](https://jupyterbook.org) e [JupyterLite](https://jupyterlite.readthedocs.io).
