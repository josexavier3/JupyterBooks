# Fatigue Design of Mechanical Joints and Aerospatial Structures

**Department of Mechanical and Industrial Engineering** · Faculdade de Ciências e Tecnologia | FCT NOVA

A collection of worked problem sheets on fatigue and fracture mechanics, with Python
programming integrated in *Jupyter Notebook / JupyterLab* and run in the browser via
**Google Colab**.

## How to navigate

::::{grid} 1 1 2 2
:gutter: 3

:::{grid-item-card} Problem sheets
Pick a problem sheet in the sidebar. Each sheet gathers the problems for that session.
:::

:::{grid-item-card} Run in Colab
Each sheet links to a Google Colab notebook with the full solutions — run and edit the
Python code in the browser, nothing to install.
:::
::::
